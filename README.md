# group_2

## **Barbie ATM**

### 📖 Kuvaus

Rakennetaan Barbie- teemainen pankkiautomaatti, jolla pystyy
seuraamaan kortin sisäisiä pankkitilejä. Pankkiautomaatti on suunnattu 
nuorilla naisille, jotka ovat kyllästyneitä samaan harmaaseen pankkiautomaattiin.

### 🚀 Käyttöohjeet

Asennus

Asenna tarvittavat riippuvuudet:

npm install

tai

pip install -r requirements.txt

Käynnistys

Käynnistä sovellus:

npm start

### ✨ Ominaisuudet

✅ Kaksoiskortti ominaisuus, käytä credit ja debit

✅ Credit-kortti ominaisuus, 3 eri tasoista credit-korttia

⏳ Säästötili, vanhemmat voivat luoda säästötilin lapsilleen, josta
ei voi nostaa rahaa. 5- vuoden kuluttua rahanmäärän kasvaa vuosittain 2%.

### 🛠 Teknologiat

Ohjelmointikieli: Javascript, C++

Käytetyt kirjastot ja frameworkit: Express, Node
